Problem Statement
Many people want to improve their typing speed and accuracy but lack a quick and easy tool to self-assess their performance. This project addresses the need for a simple, accessible way to measure typing speed and accuracy in real-time.

Scope of the Project
The project provides a minimalistic command-line typing speed test where users type a predefined sentence. It calculates the time taken, words per minute (WPM), and accuracy percentage to give users immediate feedback.

Target Users
Individuals practicing to improve typing speed and accuracy.

Beginner programmers learning Python.

Educators and trainers who want a simple typing test tool.

High-level Features
Fixed sample sentence for typing.

Measurement of time taken, WPM, and accuracy.

Immediate feedback after typing.
